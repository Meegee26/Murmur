# Murmur

A real-time messaging application built with modern web technologies, enabling users to communicate instantly with seamless message delivery and live updates.

## Features

- **Real-time Messaging** - Send and receive messages instantly
- **User Authentication** - Secure sign up and login with password encryption
- **Live Updates** - Real-time message synchronization across clients
- **User Management** - View and manage user profiles
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## Tech Stack

#### Frontend

- **React** - UI library
- **Vite** - Build tool and dev server
- **Mantine** - React component library for building UIs
- **JavaScript (ES6+)**

#### Backend

- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **JavaScript (ES6+)**
